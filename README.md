# Documentación del Proyecto de Monitoreo y Gestión

Este repositorio contiene la documentación y archivos de configuración relacionados con Elasticsearch, Kibana, Logstash y la monitorización de máquinas virtuales (MVs).

---

## Estructura de Directorios

### 1. Elasticsearch
Contiene todo lo necesario para configurar y administrar el cluster de Elasticsearch.

- **Archivos de configuración del cluster**: Carpeta con los elasticsearch.yml de cada nodo
- **apikeys.json**: Archivo con claves de acceso y API keys.
- **Documentacion Elasticsearch.pdf**: Documentacion de todo lo realizado en elasticsearch y prueba de redundacia de los nodos.

---

### 2. Kibana
Documentación y configuración de Kibana.

- **Documentacion Kibana.pdf**: Documentacion de los dashboards y alertas realizadas en kibana.
- **kibana.yml**: Archivo de configuración de Kibana.

---

### 3. Logstash
Archivos y documentación para la configuración de Logstash.

- **config-scrapper**: Carpeta con las configuraciones para ingesta de datos.
- **Documentacion Logstash.pdf**: Documentacion y explicacion de los archivos de configuracion de ingesta de datos.
- **logstash.yml**: Archivo principal de configuración de Logstash.
- **pipelines.yml**: Configuración de pipelines para procesamiento de datos.

---

### 4. Monitorización de las MVs
Documentación sobre cómo se realiza la monitorización de las máquinas virtuales.

- **Documentacion de la monitorizacion de MVs.pdf**: Documentacion del uso de metricbeat y los paquetes usados.

---

### 5. Documentacion General
Documentación sobre los pasos seguidos para el despligue del cluster
